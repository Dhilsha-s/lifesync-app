# LifeSync — AI Productivity App

An AI-powered productivity and accountability app built with React, 
Tailwind CSS, Groq AI, and Supabase.

## Features
- 🤖 AI goal breakdown (Year → Month → Week → Day) using Groq LLaMA
- ✅ Daily task tracker with streak counter
- ⏱️ Focus timer with Pomodoro sessions
- 📊 Analytics dashboard with weekly insights
- 💾 Real-time data saved to Supabase
- 🔒 Rate limiting and input validation
- 🎨 Responsive design with glassmorphism UI

## Tech Stack
- React + Tailwind CSS
- Groq AI API (LLaMA 3.3)
- Supabase (PostgreSQL)
- Vercel (deployment)
